I wanted to share a little bit about what I learned during the fall semester of 20akldjkf21.

I took four classes during the fall semester:

  • Technical and Professional Writing
  • Rhetoric, Technology, and the Internet
  • Introduction to Algorithms and Data Structures
  • Discrete Structures of Computer Science

I took two classes for my major (computer science) and two classes for my minor (technical writing and communication). I'd like to share some highlights of what I learned in each class.

Introduction to Algorithms and Data Structures

I would say this was the class I learned the most in. The class provides an overview on some of the basic concepts in algorithms and data structures such as linked lists, stacks/queues, binary trees and binary search trees, merge sort, binary search, etc. I definitely learned a lot about some of the fundamental concepts of CS. Some of the projects included building a linked list from scratch and creating maze solving algorithms.

I also was introduced to the Java language in this course. I had only had experience with dynamically typed languages such as Python and JavaScript before this, so it was fun to work with a statically typed language. I also learned a lot more about object oriented programming. Learning Java was really fun! Next, I'm gonna be learning C. It'll be interesting to see how C differs from Java.

Discrete Structures of Computer Science

In this class, some of the more theoretical concepts of CS were covered. It was more of a math class than a CS class, there wasn't any programming involved. I learned a lot of things in this class. Some of the topics were set theory, proofs (especially induction proofs), functions/relations, and counting problems/probability. I enjoyed this class because it made me see CS in a different way, and it also helped me see different ways of solving problems. This was definitely a difficult class, but a rewarding one as well.

Technical and Professional Writing

I learned to write professional and technical documents in this class. The class centered around 3 writing assignments: a technical definition, a memo detailing two interviews, and a ten page proposal. For my technical definition, I made a document defining the term API. That was a fun one! I honed my writing skills in this class and practiced writing meaningful technical documents.

Rhetoric, Technology, and The Internet

This was more of a discussion based class. We discussed a lot of topics in rhetoric and technology, such as racially biased algorithms and concerns of privacy and surveillance on the internet. It was enlightening to learn more about current problems in technology today.

Overall, I'm happy that I got to learn a lot of new things and I'm excited to see what's next! Next semester I'll be taking courses in linear algebra, machine architechture and organization, and algorithms and data structures.
