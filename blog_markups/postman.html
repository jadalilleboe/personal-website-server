Postman is a developer tool/platform for testing and creating API's. It's very useful when building a new API or testing an existing one. Postman also has a very vibrant student community, which I became involved with a few months ago. I'd like to chronicle my experience with the Postman student community and tell you how you can get involved.

I was first introduced to Postman when an API's 101 workshop hosted by Postman was held at my university. In this workshop, I learned the basics of sending requests to an API and getting a response back. I also learned how the Postman interface works and how to send a request with Postman. It was a fun experience learning about the basics of API's.

After that initial workshop, I was really interested in learning more about how API's work and how I could create my own API. I used resources such as fullstackopen.com and FreeCodeCam[ to learn how to build and API. I ended up making an API with data from my favorite show, Survivor. I used Postman to test this API and also organize all of the requests that could be made to my API.

After doing a little more work with Postman, I discovered that they had two programs for students: Postman Student Expert and Postman Student Leader. The Postman Student Expert is a training that helps you learn more about how Postman is best utilized. I learned how to set collection and environment variables, use custom scripting, and how to use Postman Visualizer to visualize JSON data from API's. After completing the training, I got a cute little badge that I could put on my LinkedIn.

Soon after receiving my Postman Student Expert badge, I applied to be a Postman Student Leader. Postman Student Leader's advocate for API literacy on their campus by holding API workshops, attending conferences, etc. I'm happy to say that I was accepted to the program!
